Hier wird ausgewählt, wie das Build aus dem Artefakte kopiert werden sollen, selektiert wird. Zum Beispiel das letzte erfolgreiche und stabile Build oder das letzte "Keep Forever" Build.
Andere Plugins können weitere Auswahlmöglichkeiten hinzufügen.
Die Buildnummer des ausgewählten Builds wird in einer Umgebungsvariable als Referenz für weitere Buildschritte gespeichert. Der Name der Umgebungsvariable ist COPYARTIFACT_BUILD_NUMBER_ mit dem Namen des Quellprojekts angehängt (alles in Großbuchstaben, alle Zeichen außer A-Z werden durch Unterstriche ersetzt).
